# Гонки

Небольшая браузерная игра-гонка на JavaScript.

## Управление
- Влево: `←` или `A`
- Вправо: `→` или `D`
- Пауза: `Space`
- Старт/рестарт: `Enter`

## Особенности
- Экран выбора машины
- Разные препятствия: легковая, грузовик, конус, масло
- Ускорение по времени и очкам
- Рекорд сохраняется в `localStorage`
- Звуки старта, очков, масла и столкновений

## Запуск
Открой `index.html` в браузере.

## Файлы
- `index.html`
- `style.css`
- `script.js`
